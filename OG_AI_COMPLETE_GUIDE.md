## Setup Instructions

### Quick Start

1. **Run the setup batch file:**

   ```
   SETUP_OG_GANGSTER.bat
   ```

   This installs all dependencies automatically.

2. **Configure AI Providers (Optional but Recommended):**

   Create a `.env` file with your API keys:

   ```
   # OpenAI (optional)
   OPENAI_API_KEY=your_openai_key_here
   OPENAI_MODEL=gpt-4o-mini
   
   # Anthropic Claude (optional)
   ANTHROPIC_API_KEY=your_anthropic_key_here
   ANTHROPIC_MODEL=claude-3-5-sonnet-20241022
   
   # Ollama (local, free)
   OLLAMA_MODEL=llama3.2
   
   # Features
   GHETTO_MODE=true
   GANGSTER_MODE=true
   SWEARING_ENABLED=true
   VOICE_ENABLED=true
   ENABLE_SELF_LEARNING=true
   ```

3. **Install Ollama (For Local AI - FREE):**

   ```
   # Download from: https://ollama.ai/download
   ollama pull llama3.2
   ```

   Keep Ollama running in background for local AI.

4. **Start OG-AI:**

   ```
   python og_supreme_agent.py
   ```

   Or double-click: `TEST_OG_AI.bat`

- Run: `SETUP_OG_GANGSTER.bat`

- Keep it running in background

- Windows: Should work automatically
- Linux: `sudo apt-get install espeak`
